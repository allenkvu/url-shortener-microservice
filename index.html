API: URL Shortener Microservice

By Allen Kenta Vu Project GitHub

Example usage:

https://akvu-url-shortener.herokuapp.com/new/http://www.google.com

Example output:

{"original_url": "http://www.google.com","short_url": "HkFLAXDQx"}


Now https://akvu-url-shortener.herokuapp.com/HkFLAXDQx will automatically navigate to www.google.com!
